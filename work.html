PROJECTS

DATA WAREHOUSING: We were asked to provide a data warehousing prototype for a company called Helius LLP. Helius is a medium sized company and uses an OLTP system with over 40 tables. Our first job was to design and populate a data warehouse that supports this OLTP system. Then, we have to give queries and generate OLAP reports using PENTAHO reporting and KETTLE for ETL process. We used Stored procedures and C# programs to populate the tables .https://github.com/rajat26/Data-Warehousing

PASSWORD CRACKING: We analyzed four open source password cracking software namely: Ophcrack, Cain and Abel , SamInside and Proactive password Auditor and compared their timings of cracking a user's defined password. We also developed our own password cracking software which can crack passwords through brute force or dictionary and we compared the results with the other tools. https://github.com/rajat26/Password-Cracker--PHP

DATA SANITIZATION THROUGH K-ANONYMITY: In this project, we explored the concept of K-anonymity by generalizing the real-life data and then we compared the accuracies of generalized data and the normal data with the help of rapid miner tool by choosing one of the attribute as a class label.

CRYPTOGRAPHICALLY SECURE RNG FOR PHP LIBRARY IN LINUX: In the recent paper " I Forgot Your Password: Randomness Attacks Against PHP Applications", it was proved that the random functions of PHP are not secured . So we built a secured random number generator function using dev/urandom and compared the results by calculating entropy using tools like ENT and NIST-STS. https://github.com/rajat26/Patch-for-PHP-random-number-Generator
